# Sistema-Operacional-Fake-Web

Este projeto foi criado inicialmente apenas para **gravar um vídeo**.
Sim, parte do sistema foi **gerada com ajuda de IA**, porque a ideia original era só demonstrar algo rápido para o vídeo.

No final, o plano não saiu exatamente como eu queria. Mesmo assim, eu acabei pegando esse sistema, **modificando algumas partes, corrigindo outras e experimentando algumas ideias**, e foi assim que esse projeto acabou ficando do jeito que está hoje.

Sendo bem sincero: **o projeto ainda está bem bagunçado** e longe de algo realmente pronto ou profissional. Este repositório existe mais como um **registro da ideia** para talvez melhorar no futuro.

Provavelmente eu vou **apagar essa versão do meu computador**, mas deixei aqui no GitHub para guardar o conceito.
